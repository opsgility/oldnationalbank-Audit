## How to use
Each exercise is self-contained: objective, prerequisites, step-by-step prompts, a deliverable, and a verification checklist. All prompts are copy-paste ready. Work in **Excel for the web** with the sample workbooks saved in OneDrive.

## Ground rules
Copilot accelerates the mechanics; the auditor concludes. **Work on a copy** — Copilot edits directly and auto-saves (there is no Track Changes in Excel; version history is your safety net). Copilot only sees the **open workbook**, and needs clean table-shaped data: one header row, no merged cells, no subtotals, no blank rows/columns, AutoSave on, automatic calculation. Tie out control totals after every Copilot build. Use only synthetic sample workbooks inside ONB's Microsoft 365 tenant.
